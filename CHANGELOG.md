# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-01

### Added
- Initial release
- JSON parsing and serialization (`parse_json`, `to_json`)
- CSV line parsing (`parse_csv_line`)
- Email validation (`validate_email`)
- Phone number validation (`validate_phone`)
- Dictionary utilities (`flatten_dict`, `merge_dicts`, `filter_dict_keys`)
- Unit tests for all functions
- Documentation and examples
